# Sílvia Banda Enigma - Landing Page

Landing page institucional e portfólio para a musicista Sílvia Banda Enigma.

## Tecnologias
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion (Motion)

## Instalação
```bash
npm install
```

## Desenvolvimento
```bash
npm run dev
```

## Build
```bash
npm run build
```
